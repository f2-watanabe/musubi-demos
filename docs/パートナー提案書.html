紹介パートナーのご案内 — Musubi

貴社のお客様に「集客できる公式サイト」を。紹介だけで、継続報酬をお支払いします。

Musubi とは

飲食店・中小企業向けのWebサイト制作・運用サービスです。制作して終わりではなく、公開後も運用まで一括で支えます。

  • 公式サイト制作（スマホ最適化・最短2週間）
  • ネット予約（手数料0円）・LINE予約
  • Google口コミへのAI返信・月次レポート
  • 顧客自身でのセルフ編集は回数無制限

料金: 飲食店向け 初期5万〜15万円+月額5千〜2万円 / 企業向け 初期9.8万〜39.8万円+月額8千〜2.5万円。長期縛り・違約金なし（いつでも解約可能）。

紹介いただきたいお客様

  • 食べログ等には載っているが、公式サイトを持っていない飲食店
  • Googleの口コミ評価は高いのに、ネット予約の受け皿がないお店
  • ホームページが古いまま放置されている中小企業

パートナー報酬

項目内容
報酬初期費用の10% + 月額利用料の10%を12ヶ月間
例: 飲食スタンダード（初期10万+月1.5万）10,000円 + 1,500円×12ヶ月 = 28,000円/件
例: 企業コーポレート（初期24.8万+月1.5万）24,800円 + 1,500円×12ヶ月 = 42,800円/件
支払月末締め翌月末払い・振込手数料は当社負担

パートナーにしていただくこと

ご紹介のみです。お客様をおつなぎいただければ、商談・ヒアリング・制作・公開後のサポートまで、すべて Musubi が対応します。貴社の業務負担はありません。

  1. 「サイトを作りたい/集客に困っている」お客様に Musubi をご紹介
  2. お客様の同意を得て、当社へご連絡（お名前と連絡先のみ）
  3. 成約・決済確認後、報酬をお支払い
