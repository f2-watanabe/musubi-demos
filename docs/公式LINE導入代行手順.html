公式LINE導入代行手順（プレミアムプラン）

Musubi プレミアムプランの柱「公式LINEまるごと導入代行」を、契約店舗ごとに Musubi運営スタッフが代行してセットアップするための手順書。

店舗はLINE公式アカウントの開設から普段の運用イメージ、予約の自動化までを丸ごとお任せでき、Musubiが①アカウント開設 → ②基本設定 → ③リッチメニュー → ④友だち追加導線 → ⑤LINE自動予約の接続 → ⑥店舗への引き渡し、の順に代行する。

1. LINE公式アカウントの開設（店舗名義）

  1. LINE公式アカウント の申込ページから、店舗名義でアカウントを開設する。アカウント名は店名（例：さくら亭）で登録する。
  2. 業種・所在地など基本情報を入力する。認証済みアカウント（バッジ付き）にするかどうかは任意 — 審査に数日かかるため、急ぎの場合は未認証のまま先に進めてよい（後から認証申請しても問題ない）。
  3. 開設後、LINE Official Account Manager にログインできることを確認する。
  4. 「LINE自動予約」まで接続する店舗は、このチャネルを5章の手順で Messaging APIチャネル化する（LINE自動予約セットアップ手順.md の2章を参照）。

2. 基本設定（あいさつ・応答・チャット）

manager.line.biz の対象アカウントで、次を設定する。

  1. あいさつメッセージ: 友だち追加時に届く自動メッセージを、店舗名・営業時間・予約導線を含む内容に差し替える（デフォルト文面のままにしない）。
  2. 応答設定:
  • 「応答メッセージ」は原則オフにする（LINE自動予約を接続する店舗は、Webhookより先にLINE側の定型応答が処理してしまうため必須。接続しない店舗でも、キーワード応答に頼らずスタッフが直接返す運用の方がお店の雰囲気に合うため推奨）。
  • 「チャット」はオンにする。友だちからのメッセージに、店舗スタッフのスマホアプリ（LINE Official Account Manager アプリ）で直接返信できる状態にする。
  1. 上記2点は、あとから店舗側で変更可能であることを引き渡し時に伝える。

3. リッチメニュー作成（3ボタン構成の雛形）

トーク画面下部に常時表示される「リッチメニュー」を作成する。基本は次の3ボタン構成を雛形として使う。

ボタン遷移先備考
予約する店舗の予約ページ（reserve.html）一番目立つ位置（左 or 中央）に配置
メニューを見る店舗の公式サイト内メニューページ写真付きメニューがあれば優先してリンク
地図・アクセスGoogleマップの店舗ページ電話番号タップも兼ねる場合はここに併記
  1. manager.line.biz の「リッチメニュー」から新規作成する。
  2. デザインは店舗の公式サイトと色味を合わせる（--green --amber 等のブランドカラーをテンプレートに反映）。ゼロから作る場合は3分割の横並びテンプレートを使う。
  3. 各ボタンのタップ動作を「リンク（URL）」に設定し、上表の遷移先を貼る。
  4. 保存して「使用中」に切り替え、実際にスマホのLINEアプリで表示・タップ動作を確認する。

4. 友だち追加QRの取得と店頭POPデータ作成

  1. manager.line.biz の「友だち追加ガイド」からQRコード画像をダウンロードする。
  2. 店舗の公式サイトと合わせたデザインの店頭POP（A4またはA5、卓上スタンド用）を作成する。含める要素:
  • QRコード（大きめ・十分な余白）
  • 「友だち追加で予約がかんたんに」等の一言コピー
  • 店舗ロゴ・店名
  1. 完成したPOPデータ（PDF・PNG等）を店舗へ納品データとして渡す（印刷は店舗側で手配、印刷代行が必要な場合は別途相談）。

5. LINE自動予約の接続（プレミアムのみ）

公式LINEまるごと導入代行の最終ステップとして、予約ページのLINEボタンを「送るだけで自動確定」の自動予約モードに切り替える。

具体的なMessaging API化・Webhook接続・動作確認の手順は、既存の LINE自動予約セットアップ手順.md に従う（本手順書では重複させない）。この章では、公式LINE導入代行の一連の流れの中で「どのタイミングで実施するか」だけを示す。

  1. 1章でチャネルを作成した時点で、Messaging APIチャネル化（LINE自動予約セットアップ手順.md 2章）まで済ませておくと、5章以降の作業がそのまま流用できる。
  2. アカウント開設・基本設定・リッチメニュー・QR/POPの4章までが完了したら、LINE自動予約セットアップ手順.md の3章（Musubi adminへの登録）〜6章（動作確認チェックリスト）を実施する。

6. 店舗への引き渡しチェックリスト

[ ] LINE公式アカウントが店舗名義で開設され、manager.line.biz にログインできることを店舗担当者と一緒に確認した
[ ] あいさつメッセージが店舗名・営業時間・予約導線を含む内容になっている
[ ] 応答メッセージがオフ、チャットがオンになっている
[ ] リッチメニューが公開され、予約ページ／メニュー／地図の3ボタンが正しく機能する
[ ] 友だち追加QRのPOPデータを店舗に納品した
[ ] LINE自動予約を接続する場合、LINE自動予約セットアップ手順.md の動作確認チェックリストをすべて完了した
[ ] 店舗スタッフのスマホにLINE Official Account Managerアプリを案内し、チャット返信ができることを確認した
[ ] manager.line.biz のログインID・パスワードを店舗側にも共有した（アカウントの所有権は店舗にあることを明示）

全項目確認できたら引き渡し完了。以降のリッチメニュー更新やあいさつメッセージの変更は、月額保守の更新対応の範囲内で対応する。
