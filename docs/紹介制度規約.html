Musubi 紹介制度 利用規約

第1条（目的）

本規約は、Musubi（以下「当社」）が提供するWebサイト制作・運用サービス（以下「本サービス」）の紹介制度（以下「本制度」）について、紹介者と当社の間の条件を定めるものです。

第2条（定義）

  1. 「紹介者」とは、本規約に同意のうえ本制度に参加する個人または法人をいいます。
  2. 「被紹介者」とは、紹介者の紹介により本サービスを申し込む店舗・事業者をいいます。
  3. 「成約」とは、被紹介者が当社と本サービスの契約を締結し、初回の月額利用料の決済が完了したことをいいます。

第3条（参加資格）

  1. 本制度には、当社の顧客であるか否かにかかわらず参加できます。業者・法人の方も本制度で参加できます。
  2. 未成年者、反社会的勢力に該当する者、過去に本規約違反により資格を取り消された者は参加できません。

第4条（紹介の成立）

  1. 紹介は、当社所定の方法（紹介コードの利用、申込時の紹介者名の申告、または当社への事前連絡）により行うものとします。
  2. 次の場合、紹介は成立しません。
  • 被紹介者が申込時点で既に当社と商談中または契約中の場合
  • 紹介者自身による申込（自己申込）の場合
  • 紹介の事実が確認できない場合

第5条（報酬）

  1. 報酬は、被紹介者の月額利用料（税抜）の30%を、初回決済月から6ヶ月間支払います。
  2. 被紹介者が6ヶ月以内に解約した場合、解約月の翌月以降の報酬は発生しません。
  3. 無料期間・トライアル期間中の解約、および決済が完了しなかった月は報酬の対象外です。
  4. 初期費用は報酬の対象外です。

第6条（報酬の確定と支払）

  1. 報酬は、被紹介者の決済完了月の月末に締め、翌月末に紹介者の指定口座へ振り込みます。
  2. 最低支払額は5,000円とし、未達の場合は達するまで翌月以降に繰り越します。
  3. 振込手数料は当社が負担します。
  4. 報酬の受領に伴う税務処理（確定申告等）は紹介者の責任で行うものとします。

第7条（禁止事項）

紹介者は次の行為をしてはなりません。違反した場合、当社は報酬の不支給・参加資格の取消しができます。

  1. 虚偽または誇大な説明による勧誘（効果・実績の捏造を含む）
  2. スパム行為（無差別なメール送信・SNS投稿等）
  3. 当社または第三者になりすます行為
  4. 被紹介者に金銭その他の利益を提供して申込を誘引する行為（当社が公認したキャンペーンを除く）
  5. 当社の信用を毀損する行為

第8条（制度の変更・終了）

  1. 当社は、1ヶ月前までの通知により本制度の内容を変更し、または終了できます。
  2. 変更・終了の時点で既に確定している報酬は、変更後も従前の条件で支払います。

第9条（その他）

  1. 本制度への参加により、当社と紹介者の間に雇用・代理・委任関係は生じません。
  2. 本規約に定めのない事項は、当社と紹介者が誠実に協議して定めます。

第10条（本サービスにおけるAIの利用）

  1. 紹介者が案内する本サービスには、被紹介者（店舗）宛てのGoogle口コミへの返信文を、AIが下書き生成する機能が含まれます。
  2. 生成された下書きは自動投稿されるものではなく、被紹介者が内容を確認・承認したうえで投稿する運用です。
  3. 紹介者は、勧誘にあたり本サービスの内容（AIによる下書き生成を含む）を正確に説明するものとし、実態と異なる誇大な説明をしてはなりません（第7条第1号）。
